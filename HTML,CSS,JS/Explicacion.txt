En estas carpetas se encuentran los proyectos en sus diferentes etapas realizados en la materia "Web 1",el cual consistia en realizar una pagina utilizando HTML y CSS,en la segunda parte se comenzo a incorporar el comportamiento y logica de la misma utilizando JS y generar una tabla la cual debia ser rellenada consumiendo una API Rest(en este caso MockAPI),para la tercer parte del mismo,el objetivo era hacer que esta misma sea totalmente responsive sin la utilizacion de tecnologias como Boostrap,el cual fue logrado utilizando las media queries ya incluidas en CSS.
